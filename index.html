Features

What this boilerplate gives you out of the box.

  • 📂

    Zero dependencies

    No npm, no bundler, no framework. Open the folder in your browser and it works.

  • ✅

    Accessible by default

    Semantic HTML, visible focus styles, skip links, ARIA where appropriate, and reduced-motion support.

  • 📱

    Responsive layout

    Mobile-first CSS with a fluid container, flexible grid, and no hardcoded breakpoint assumptions.

  • 🌐

    PWA-ready (optional)

    Optional manifest and service worker are included. Enable them when you're ready for offline support.

  • 🔄

    GitHub Pages friendly

    Deploy by pushing to a branch. All paths are relative. Includes a 404.html for SPA-style fallback.

  • ✎

    AI-extendable

    Structured and commented for easy collaboration with AI assistants. See the prompting guide in README.
